1) What is the difference between null and undefined?
null means a variable has no value. 
undefined means a variable is declared but currently not assigned a value.

5) Explain the concept of Scope in JavaScript (Global, Function, Block).
Global Scope: variable can be used from anywhere.
Functional Scope: variable can be only used inside the function it is declared.
Block scope: varibles can only be used inside the block where it is declared.
